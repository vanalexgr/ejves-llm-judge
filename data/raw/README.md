# Raw Input Note

This directory is a placeholder describing the historical source materials used to reconstruct the processed study dataset.

The public repository does not ship a full raw-data bundle of the archived reviewer workbooks. Downstream reproducibility in this repository is based on the processed outputs in [data/processed](/home/vga/Documents/New project 5/data/processed), which contain the reconstructed long-format reviewer scores and consensus tables used by the calibration and comparator analyses.

Historically, the reconstruction step depended on archived reviewer spreadsheets equivalent to:

- Mario workbook
- WD workbook
- EG workbook

Those historical workbook names also appear in the repository root as archival source files used during reconstruction. Future replicators who need to repeat the full raw-to-processed reconstruction should obtain the same reviewer workbooks and then run the Phase 1 loader.
